##This is a project in progress, working towards identifying molecules from their infrared spectrometries.

To run the current version - Fourier Transformation of Image Input.py, the python file, the HTML and JSON files must all be in the same folder. The Python file has the backbone of all logic inside of it, whilst the HTML file dispalys plots for all the tested frequencies during the visual representation, with the JSON providing the data from the python file.
Currently the python file has different versions of visual representations, primarily using matplotlib for graphs of the wave - and an optional function to display the tested frequencies in matplotlib, but the HTML file is advised.
